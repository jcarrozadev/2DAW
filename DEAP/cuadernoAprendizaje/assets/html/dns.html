Introducción a DNS

DNS (Domain Name System) es un sistema de nomenclatura jerárquico descentralizado para dispositivos conectados a redes IP como Internet o una red privada. Asocia información variada con nombres de dominio asignados a cada uno de los participantes.
Utiliza el puerto UDP 53.

¿Para que nos servirá?

Su función principal es traducir (resolver) un nombre de dominio a una dirección IP.
